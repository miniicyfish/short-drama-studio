# 片场开拍机制与文字剧情 Review 稿 v2

日期：2026-05-09
状态：根据反馈重写，待 review，尚未执行代码修改

## 这版要修正什么

上一版的问题是：把演员事故写成了“人设贴标签”。比如某个演员是什么身份，就在拍摄时不断强调他的个人符号。这会变得机械，而且不一定好笑。

这版的核心改成：

> 大家都知道这个豪门竖屏短剧很颠，也知道这个剧组很草台，但场地租了、机器开了、盒饭订了、老赵把人叫来了，所有人只能硬着头皮把荒唐剧情拍下去。

喜剧来自四层错位：

- 剧本很豪门，但现场很穷。
- 台词很疯，但演员本人觉得这东西没法正常理解。
- 导演想要“爆”，但预算和时间只允许“差不多能剪”。
- 每个人都觉得这活儿离谱，但每个人又都有一点非拍不可的理由。

演员个人符号仍然重要，但它不应该被手写进每一句固定稿。具体某个演员在某个角色上会带出什么个人符号，应该在开拍前由 AI 根据“演员 + 预定角色 + 说服方式”生成。

## 机制改法

### 1. 开拍前生成“演员入戏偏差卡”

说服结束后，进入开拍前，AI 为每个演员生成一张 `performanceProfile`。

这张卡不是直接给玩家展示的设定表，而是后续拍摄生成的依据。

字段建议：

```ts
{
  actorId: "qiu-peng",
  actorName: "邱鹏",
  roleId: "zhou-assistant",
  roleName: "周特助 / 顾家长辈",
  roleUnderstanding: "他把周特助理解成一个替老板求情、其实也在替自己求一个位置的人",
  shortDramaConfusion: "他不理解为什么助理要替老板跪到这种程度，但觉得既然来了就得显得有用",
  cannotSayNoReason: "他最近太需要一份看起来像机会的活",
  personalLeak: "台词碰到没用、被抛弃、求别人留下时，会露出自己的现实情绪",
  lowBudgetFriction: "怕跪地弄脏裤子，也怕临时加动作没加钱",
  likelyAccidents: [
    "把求少夫人回家演成自己求一份工作",
    "台词越说越真，导致现场突然尴尬",
    "对下跪动作犹豫，先问地干不干净"
  ],
  usefulDirection: "如果导演把他往职业助理上拉，他会更稳；如果放任，他会变成现实苦情独白"
}
```

关键变化：这张卡不是“固定人设”，而是“这个演员如何理解这份荒唐工作”。

### 2. 开拍稿要写“片场反应”，不是写“人设展示”

不要写：

- `[女主演员]把“有脸”两个字咬得很清楚，像准备让观众做成表情包。`
- `[男主演员]因为是夜场男模，所以把霸总演成夜场。`

应该写：

- 她念到“这一口，是你欠顾家的”时停了半秒，看向导演，像是在确认这句真的要说出口。
- 顾沉舟演员低头准备强吻，动作到一半停住，小声问：这段真亲还是借位？
- 老赵在监视器后压低声音：借，当然借。你以为我们有保险吗？
- 温雅演员听见“今晚谁能留下，谁就是顾家的女主人”，先笑了一下，笑完发现没人笑，只好硬把笑收回去继续演。

这里的重点不是“她是什么人”，而是“一个正常人被迫拍疯剧本时的现场反应”。

### 3. 默认未干预版本也要有失控，但不是全面炸场

默认拍摄应该是：

- 剧本功能完整跑完。
- 每幕有 2-4 个片场反应点。
- 这些反应点让玩家感觉“这帮人真的在拍一个很颠的短剧”。
- 但事故不能太大，大事故留给玩家工具干预后升级。

默认状态像“勉强能剪，但处处漏风”。

工具干预后才进入：

- 演员真的不理解。
- 现场真的笑场。
- 台词被改到后续全歪。
- 某个演员把角色理解成自己的现实问题。
- 片场状态开始掉得明显。

### 4. 老赵的功能

老赵不是系统讲解员，也不是弹幕吐槽机。他的功能是“草台制片现实”。

他说话应该永远带着这几种压力：

- 时间不够。
- 钱不够。
- 场地不让重来。
- 演员不是专业的，别把人弄跑。
- 这段虽然离谱，但可能能剪。

好用的老赵句式：

- 别问合不合理，问能不能剪。
- 先拍，理解这种事后期再说。
- 他不是演错，他是错得有镜头价值。
- 小点声，房东以为我们真在豪门吵架。
- 借位，当然借位。我们没有保险。
- 这句别解释，越解释越像诈骗。

## 开拍文字剧情 Review 稿 v2

下面这版仍然是 9 幕默认未干预开拍稿，但不写具体演员人设。实际接入代码后，AI 会根据玩家选的人和说服结果，把 `[沈知意演员]`、`[顾沉舟演员]` 等替换成更具体的现场反应。

这版重点是“粗制滥造低成本剧组硬拍豪门疯剧本”的氛围。

### 幕 01｜顾家晚宴外廊羞辱

镜头：顾家晚宴外廊灯火很满，金色门框、假花、租来的水晶灯一起撑出一种很贵但只够拍半天的体面。

动作：群演站在两边，有人还没弄清自己是顾家亲戚还是宾客，只知道不能看镜头。

老赵：都别往镜头里找自己。你们是豪门，不是春晚观众席。

镜头：沈知意走进来，裙摆刚碰到光，现场安静了一秒。

顾家长辈：还知道回来。顾家这些年，倒是第一次为了一个外人这么热闹。

动作：顾家长辈演员念完，自己也像被这句酸了一下，眼神往导演那边飘。

沈知意：不是你们叫我来的么。现在我来了，又摆这副脸给谁看？

动作：沈知意演员接得很快，但接完停了一下，像是在等对方解释“为什么豪门要在走廊里堵人”。

老赵：别等逻辑，豪门剧逻辑在第二集才补。

镜头：顾沉舟从人群里走出来。皮鞋擦得很亮，地毯却有一角翘起来。

顾沉舟：知意，跟我进去。

动作：顾沉舟演员伸手到一半，突然意识到这句像物业调解，手又收回去一点。

沈知意：顾总是不是忘了，我三年前就不是顾家的人了。

温雅：是不是顾家的人，不是你一句话说了算。今晚能站在这里的人，才算。

动作：温雅演员说完“站在这里”，下意识看了一眼脚下胶带定位，像突然发现自己真的只是站在一个黄色叉上。

沈知意：那你站得挺稳的。

镜头：温雅脸上的笑僵住。这个僵住一半是演的，一半是真的不知道怎么接。

顾沉舟：我最后说一遍，跟我进去。

沈知意：我要是不呢？

顾沉舟：由不得你。

动作：现场安静。不是被霸总震住，是大家都听出来下一幕要开始不正常了。

老赵：好，别讨论，直接过。越讨论越像我们写错了。

### 幕 02｜顾沉舟公开越线

沈知意：三年前你让我滚的时候，也说过这句。

动作：沈知意演员念“三年前”时有点卡。不是忘词，是这句突然认真得和前面那堆假花不太匹配。

顾沉舟：那你就该知道，我说到做到。

动作：顾沉舟上前，一把拽住沈知意手腕。

动作：手刚碰上，两个演员都停了半秒，同时看向导演。

顾沉舟演员：这里真拽吗？

老赵：轻拽。拍出来像重拽。我们主要靠音效。

沈知意：你干什么——

镜头：顾沉舟把她拽到面前，现场群演集体吸了一口气，吸得太齐，像排练过。

动作：原剧本该是当众强吻。顾沉舟演员低头到一半，明显慢了。

顾沉舟演员：借位吧？

老赵：当然借位。我们没钱处理工伤，也没钱处理绯闻。

动作：两个人用一种非常努力的角度完成了一个“看起来像越线但实际隔着空气”的动作。

沈知意：顾沉舟，你有病吧？！

动作：沈知意演员这句骂得很真。真到顾沉舟演员愣了一下，差点忘了自己下一句是深情。

顾沉舟：对，我有病。从你签字离婚那天起，我就没正常过。

动作：这句太疯，现场有个群演差点笑，被旁边人用胳膊肘捅回豪门状态。

温雅：你们两个，当我是死的吗？

老赵：好。这个问题问得很合理，但角色不能合理，继续。

### 幕 03｜这一口是你欠顾家的

沈知意：你发疯别拉着我。

顾沉舟：这一口，是你欠顾家的。

动作：顾沉舟演员念完这句，空气明显空了一下。

顾沉舟演员：这一口是什么口？

老赵：别问。你一问，我们也要面对这个问题。

镜头：全场死寂。顾家长辈的表情像第一次听完整剧本。

沈知意：我欠顾家？你们顾家是有脸说这种话的？

动作：沈知意演员这次没有加情绪，只是平平说出去，反而显得这句更狠。

顾沉舟：我就是要让所有人看清楚——顾家认的人，从来都只有你。

动作：他说到“认的人”时，群演里一个演顾家亲戚的大叔轻轻皱眉，像很想问顾家到底有没有户口本。

温雅：只有她？

动作：温雅演员接得太快，快到像本人也等不及要质问这个剧情。

顾家长辈：疯了……真疯了……

老赵：这个“疯了”很好，像角色说的，也像演员说的。

沈知意：你们顾家认人，认得像盖章。

镜头：温雅往前一步，像这场荒唐终于轮到她被推上去。

### 幕 04｜温雅失控反击

动作：温雅一步一步走过去，猛地拽住顾沉舟领带。

动作：领带是服装老师临时借的，温雅演员一拽，所有人的视线都先看领带会不会断。

老赵：轻点，那条要还。

温雅：你刚才给她的，我也要。

动作：温雅把顾沉舟拉下来。顾沉舟演员顺着低头，低得太配合，像被流程培训过。

沈知意：你们顾家平时吃饭都这么演吗？

动作：这句说完，现场真的有人笑。不是角色笑，是工作人员没忍住。

老赵：谁笑的？笑可以，别笑出收音。

顾沉舟：温雅，你够了！

温雅：我够了？顾沉舟，你当着我的面吻她，现在还说我够了？

动作：温雅演员说到一半停了一下。

温雅演员：我刚才真的吻了吗？

老赵：观众觉得你吻了，你就吻了。短剧讲究心理真实。

沈知意：那你冲他去，别冲我。我今天只是路过你们顾家的精神病院。

老赵：这句谁加的？

沈知意演员：不是本里的吗？

老赵：现在是了。

温雅：如果不是你，今晚站在这里的人就是我！

沈知意：那你站啊。我又没跟你抢。

镜头：温雅站在胶带定位上，低头看了一眼那枚黄色叉，突然更气了。

顾沉舟：都给我闭嘴。

动作：这句喊完，旁边收音老师比了个手势，意思是爆了。

老赵：很好，豪门也会爆麦。

### 幕 05｜顾家选女主人

镜头：晚宴厅入口被吵声吸引，顾家长辈终于从灯里走出来。

动作：他的西装看起来很贵，但袖口上的吊牌刚被剪掉，剪得不太干净。

顾家长辈：今晚谁能留下，谁就是顾家的女主人。

动作：这句一出来，现场所有人都短暂沉默。

温雅演员：这是什么规则？

老赵：顾家的规则。

温雅演员：顾家谁定的？

老赵：编剧。

沈知意：你们顾家现在连人都要这么选？

温雅：为什么不能选？我陪了他三年，凭什么不能选？

动作：温雅演员终于找到一点能理解的情绪，结果情绪刚上来，灯光老师说她站出光了。

老赵：别出光。你再委屈也得在预算里委屈。

顾家长辈：顾家的体面不能没人撑，谁能撑住，谁就留下。

沈知意：你们要的是女主人，还是一个能替你们收拾烂摊子的人？

顾沉舟：够了，这不是你们能决定的事。

沈知意：你现在拦他们，是怕他们羞辱我，还是怕他们把你的心思说太明白？

动作：这句说完，顾沉舟演员沉默得太久。导演没有喊卡，因为这个沉默看起来像演技。

镜头：人群外传来急促脚步声。

老赵：周特助进。别真撞门，门也是租的。

### 幕 06｜周特助下跪

动作：周特助从人群后冲出来，跑得很急。

动作：他差点踩到地上的电线，硬生生把“忠心护主”跑成了“施工现场避险”。

周特助：顾总！别闹了！老爷子撑着一口气，就是想见少夫人回家！

沈知意：你叫谁少夫人？

周特助：少夫人，顾家不能没有您！

动作：原剧本写“扑通一声跪下”。周特助演员看了一眼地面，又看了一眼导演。

周特助演员：真跪？

老赵：半跪。真跪显得我们钱给够了。

动作：周特助半跪下去，姿势非常努力，但看起来像在找掉地上的隐形眼镜。

周特助：这三年顾总没睡过一个安稳觉，老爷也没真正高兴过一天！

动作：他说得太用力，自己先被这句荒唐打动了，眼眶居然真的红了。

沈知意：所以顾家睡不好，也要我负责？

镜头：全场没人敢接话。不是因为剧情压住了，是因为这句话太像正常人会问的问题。

顾沉舟：周特助，起来。

周特助：顾总，我不能起来。

老赵：他可以起来。膝盖别废，后面还有一幕。

动作：周特助保持半跪，膝盖离地一寸，忠心和现实在同一个姿势里发抖。

### 幕 07｜只有您和温小姐的嘴

周特助：少夫人，顾总这三年碰过的女人，只有您和温小姐的嘴！

动作：这句说完，现场彻底空白。

周特助演员：这句播得出去吗？

老赵：先拍。播不播是平台的事，拍不拍是我们的事。

顾沉舟：周特助。

动作：顾沉舟演员这一声叫得很轻，像不是在责怪助理，是在求他别再讲了。

温雅：周特助，你疯了吗？！

沈知意：你们顾家，是真没一个正常人了。

动作：沈知意演员这句说完，没有人觉得她刻薄。大家都觉得她终于说了句正常话。

周特助：少夫人，您要是不回去，顾家今晚就真的散了！

动作：周特助越说越投入，投入到像忘了这是顾家的事，不是自己的事。

温雅：够了！我到底算什么？！

动作：温雅演员这句情绪上来了，但旁边群演没接住，全场突然像吵架吵到没人劝。

老赵：群演给反应。你们是豪门亲戚，不是等公交。

周特助：温小姐，您不是顾家的人。可少夫人，是老爷亲口认下的。

镜头：温雅脸色白下去。沈知意没有胜利感，只有一种“这家人果然没救”的疲惫。

沈知意：你们顾家认人，怎么认得像货物入库？

动作：老赵没有喊卡。他低头在本子上写：这句可能能爆。

### 幕 08｜沈知意全员审判

镜头：沈知意看着眼前跪着的、疯着的、哭着的、盯着她的这一群人，终于冷下来。

动作：现场也跟着安静。空调声、灯架声、远处房东催时间的消息声都显得很清楚。

沈知意：一个当众越线，一个当众发疯，一个当众下跪，一个当众选女主人。

动作：她每说一个“当众”，镜头就扫过一个人。扫到周特助时，他还没完全站起来。

老赵：别动，你这个姿势值钱。

沈知意：你们顾家今晚这出戏，真够热闹的。

顾沉舟：知意，跟我回去。只要你点头，顾家的一切都是你的。

动作：顾沉舟演员说完，自己都像觉得这承诺很空。毕竟他们现在连这条走廊都只租到下午六点。

沈知意：我现在看起来，很像缺你们顾家这口饭吗？

动作：这句落下来，现场短暂地真的有点安静。

温雅演员很轻地吸了一口气，像突然意识到这场疯戏里最正常的人一直是沈知意。

顾沉舟：你可以不要我。但你不能不要本来就属于你的东西。

沈知意：你现在是想让我回顾家，还是想让我回去，替你们所有人收拾这场烂戏？

动作：老赵盯着监视器，没说话。他难得没有算这句能不能剪。

沈知意：你当众越线，是想证明你后悔了。她当众发疯，是想证明她没输。你助理跪下来求我，是想证明顾家离不开我。

镜头：她的目光从每个人脸上扫过。豪门、前妻、现任、助理，这些词突然都像临时贴上去的标签。

沈知意：可你们从头到尾，没有一个人问过——我还想不想踏进这个门。

动作：现场静了两秒。然后隔壁传来装修电钻声。

老赵：停一下。等电钻过去。这句不能被电钻抢。

### 幕 09｜真够下贱的

顾沉舟：那你说。只要你开口，我今天就把整个顾家掀了，也把你请回去。

动作：顾沉舟演员这次没有压低声线。他像终于明白这不是霸总发狠，而是一个人把话说到没路了。

镜头：沈知意停在门口，没有回头。

沈知意：顾总。

## v2 实现重点

### 开拍前 AI 要生成什么

在说服结束、正式开拍前，新增一次或合并一次 AI 调用，为每个演员生成：

- 他如何理解这个短剧项目。
- 他为什么明知道很颠还愿意拍。
- 他分到这个角色后，最不理解哪类剧情。
- 他在低成本片场最容易卡在哪。
- 他会在哪些剧情点露出自己的真实反应。
- 导演用什么方式能把他拉回戏里。

### 开拍时 AI 要怎么用这些内容

生成每幕时，不要机械套“演员身份标签”。AI 应该先判断：

1. 当前 beat 本身荒不荒唐？
2. 正常人拍到这里会不会卡一下？
3. 这个演员的 `performanceProfile` 会让他怎么卡？
4. 老赵会怎么用草台办法把它糊过去？
5. 这一切是否仍然能剪进短剧？

### 玩家应该感受到什么

玩家不应该只是在看一条豪门短剧，也不应该只是在看演员人设秀。

玩家应该感觉自己真的坐在监视器后面，看一群临时拼起来的人拍一个离谱但可能会爆的竖屏短剧：

- 有些地方很尴尬。
- 有些地方很土。
- 有些地方因为太土反而有用。
- 有些地方大家都想笑，但必须忍住。
- 偶尔有一句突然打到真的东西。
- 玩家每次工具干预，都像在决定这场草台混乱是被救回来，还是被推向更好笑的事故。

## 需要你 review 的重点

1. 这版“大家都觉得颠但硬拍”的感觉是否更接近？
2. 老赵的草台制片感是否合适，还是需要再毒一点/再少一点？
3. 默认未干预版本现在是否太像片场喜剧，短剧剧情本身是否还够强？
4. 后续代码实现时，要不要让“演员入戏偏差卡”对玩家半公开？例如只展示一句“你不知道他怎么理解了这个角色”，但不展示完整卡。
