# 片场开拍机制与文字剧情 Review 稿

日期：2026-05-09
状态：待 review，尚未执行代码修改

## 现在的问题

开拍后目前主要是在播放《豪门剧本》的固定剧情：沈知意、顾沉舟、温雅、周特助按剧本完成 9 幕。素人演员虽然在前面有“现实身份”“说服词”“入组心态”，但这些内容没有稳定地转化成片场表演事故。

具体表现：

- 演员只在说服阶段有戏，开拍后像换回了普通短剧角色。
- 入组心态只作为 prompt 背景传给 AI，没有变成每幕可触发的规则。
- 没有“演员本人误解角色”的可见表现，比如夜场男模把霸总演成营业、老板娘把顾家戏演成情感调解。
- 工具干预影响剧情，但没有明确影响某个演员的压力、误解、失控方向。
- mock 版本几乎不使用演员心态，所以本地 demo 更像固定剧本播放器。

## 推荐改法

### 1. 增加“演员表演层”

在剧本角色和演员之间增加一层 `performanceProfile`。

每个被选演员进入角色后，生成一份可用于开拍的表演偏移：

```ts
{
  actorId: "qiu-peng",
  actorName: "邱鹏",
  roleId: "zhou-assistant",
  roleName: "周特助 / 顾家长辈",
  surfaceHabit: "把每句求情都演成自己的人生低谷独白",
  misreadRule: "听到少夫人、顾家不能没有你时，会自动理解成没人需要他自己",
  triggerBeats: ["下跪", "求回家", "顾家不能没有您"],
  accidentStyle: "真疼、真委屈、忽然把台词说得像求职失败后的自白",
  recoveryHint: "如果导演喊卡，需要把他从个人低谷里拉回周特助的职业身份"
}
```

这层不是给玩家直接看的后台数据，但它决定开拍时演员怎么把角色演歪。

### 2. 初始 9 幕不是只按剧本生成，而是“剧本骨架 + 演员表演层”一起生成

当前 `buildDraftPrompt` 已经传入 casting、selectedActors、recruitResults，但指令太泛，只说“入组心态轻微影响表演”。需要改成更硬的规则：

- 每一幕至少有 2-4 行体现演员本人造成的片场戏剧感。
- 演员本人造成的戏剧感不能写成后台说明，要写成可见表演：走位不对、台词理解偏、表情抢戏、动作边界抗拒、过度投入。
- 每个角色的关键 beat 都要检查对应演员的 `triggerBeats`。
- 如果演员心态和剧本冲突，优先让剧本功能保留，但表现方式被演员演歪。

### 3. 玩家看到的是“戏中戏”，不是系统解释

不要出现：

- “因为张嘉豪是夜场男模，所以他把霸总演成夜场互动。”
- “林小满的入组心态触发了公关危机。”

应该出现：

- 张嘉豪念“知意，跟我进去”时，手下意识做了一个请进包厢的姿势，老赵在监视器后吸了一口凉气。
- 林小满听到“这一口”时停了半拍，先看镜头，再看顾沉舟，像在判断这句话会不会被截成热搜。

### 4. 片场状态与演员压力绑定

每幕拍完后，不只是写入剧情事实，也要写入演员状态：

- `pressure`：这个演员被逼到什么程度。
- `confidence`：他是不是更相信自己理解的演法。
- `deviation`：他偏离角色的程度。
- `relationshipHeat`：他和其他演员之间的现场火药味。

玩家工具不是只改剧情，而是改演员状态：

- 喊卡：提高可控，但可能伤演员自尊。
- 改词：改变剩余剧情，也改变演员对角色的理解。
- 加鸡腿：缓解压力，但可能让演员觉得“自己被买通”，从而更现实。
- 导演示范：提高明确性，但可能让演员模仿过头，把导演的尴尬也演进去。

### 5. UI 层增加“片场事故可见化”

开拍时每一幕应该有三类文本：

- 戏内剧情：沈知意、顾沉舟等角色正在演什么。
- 片场表演偏移：演员本人怎么把这句演歪。
- 导演可操作点：当前这句是否值得干预。

注意：不要把“炸点/雷点/trigger”直接显示给玩家。应该通过表演让玩家看出来。

## 开拍文字剧情 Review 稿

下面是一版“默认未干预”的片场开拍稿。它仍然保留豪门短剧 9 幕骨架，但每幕加入演员本人造成的戏剧感。这里先用角色槽位写法，实际进代码后会根据玩家选择替换为对应演员的具体表现。

角色槽位：

- 沈知意演员：`[女主演员]`
- 温雅演员：`[女二演员]`
- 顾沉舟演员：`[男主演员]`
- 周特助 / 顾家长辈演员：`[助理演员]`

### 幕 01｜顾家晚宴外廊羞辱

镜头：顾家晚宴外廊灯火很满，宾客和亲戚站在两侧，像一群提前排好队的审判员。

镜头：沈知意停在门口。她的裙摆刚进光里，现场群演已经自动往两边让开，像这不是晚宴，是村口看热闹。

动作：[女主演员]看了一眼镜头，没有立刻走。她像是在确认自己有没有被拍到不好看的角度。

顾家长辈：还知道回来。顾家这些年，倒是第一次为了一个外人这么热闹。

动作：[助理演员]把这句说得太认真，旁边一个群演差点接不上“嫌弃”的表情。

沈知意：不是你们叫我来的么。现在我来了，又摆这副脸给谁看？

内心：沈知意不是怕，是在确认顾家今晚到底想把她摆成什么东西。

动作：[女主演员]说完这句，下巴扬得过了一点。不是前妻的冷，是短视频变装结尾那种“姐回来了”。

镜头：顾沉舟从人群里走出来，皮鞋停在她面前半步。

顾沉舟：知意，跟我进去。

动作：[男主演员]伸手的姿势太熟练，像在夜场门口请人进包厢，又像在替老板迎贵客。

老赵：这手势不对，但先别喊。能剪成霸总。

沈知意：顾总是不是忘了，我三年前就不是顾家的人了。

温雅：是不是顾家的人，不是你一句话说了算。今晚能站在这里的人，才算。

动作：[女二演员]说“站在这里”时往灯下挪了半步，刚好把自己挪进最亮的位置。

沈知意：那你站得挺稳的。

镜头：温雅的笑僵住。她不是没听懂，是听懂得太快。

顾沉舟：我最后说一遍，跟我进去。

沈知意：我要是不呢？

顾沉舟：由不得你。

动作：[男主演员]这句压得很低，压出了一点不该有的营业气。现场居然有个女群演轻轻“哇”了一声。

老赵：别哇。我们这是离婚夜，不是男模返场。

### 幕 02｜顾沉舟公开越线

沈知意：三年前你让我滚的时候，也说过这句。

内心：她把顾沉舟现在的“挽回”和当年的驱逐放在一起。

动作：[女主演员]说到“三年前”时停了一下，像真的想起了什么不该想的事。

顾沉舟：那你就该知道，我说到做到。

动作：顾沉舟上前，一把拽住沈知意手腕。

动作：[男主演员]拽得不重，但距离贴得太近，近到不像霸总，像下一秒要问“姐姐喝什么”。

沈知意：你干什么——

动作：[女主演员]反应很快，先护住自己的袖口，再看镜头，像本能地怕这一下被截出来乱传。

镜头：顾沉舟把她拽到面前，现场所有人都屏住呼吸。

动作：原剧本该是当众强吻。[男主演员]低头到一半，忽然停了半秒。

顾沉舟：你躲什么？

老赵：他怎么还加词了？

沈知意：我不躲，难道站着等你发疯？

动作：[女主演员]抬手就是一巴掌，角度很准，声音很响，但离脸还有半寸。

镜头：这一巴掌像真打了。群演先愣，摄影师后退了一步。

沈知意：顾沉舟，你有病吧？！

顾沉舟：对，我有病。从你签字离婚那天起，我就没正常过。

动作：[男主演员]把“有病”说得太真诚，像在给客人解释自己为什么今晚状态不好。

温雅：你们两个，当我是死的吗？

动作：[女二演员]这句抢得太快，几乎盖住沈知意的呼吸声。

老赵：好，女二急了。这个能用。

### 幕 03｜这一口是你欠顾家的

沈知意：你发疯别拉着我。

顾沉舟：这一口，是你欠顾家的。

动作：[男主演员]说完“欠顾家”自己都皱了一下眉，像终于听见这句有多离谱。

镜头：全场死寂，连顾家长辈都愣住了一秒。

动作：[女主演员]没有马上接台词。她先看了一眼顾沉舟，又看了一眼镜头。

内心：她不是被打动，是确认顾家仍然把她当债务。

沈知意：我欠顾家？你们顾家是有脸说这种话的？

动作：[女主演员]把“有脸”两个字咬得很清楚，清楚到像准备让观众做成表情包。

顾沉舟：我就是要让所有人看清楚——顾家认的人，从来都只有你。

动作：[男主演员]说“只有你”时眼神飘了一下，飘到旁边的温雅身上，像职业习惯里不想冷落任何一个观众。

温雅：只有她？

动作：[女二演员]笑了一声。不是温雅的冷笑，是演员本人真的被这句恶心到了。

顾家长辈：疯了……真疯了……

动作：[助理演员]演长辈时忘了压年纪，语气像小区保安看见业主吵架。

老赵：长辈这个年龄感不对，但那个嫌弃劲对了。

温雅：顾沉舟，你把我放在哪里？

镜头：温雅往前一步，从旁观者变成下一幕的主动失控者。

### 幕 04｜温雅失控反击

动作：温雅一步一步走过去，猛地拽住顾沉舟领带。

动作：[女二演员]拽得太熟练，像短视频里教人“如何拿捏前任”的第一步。

温雅：你刚才给她的，我也要。

动作：她把顾沉舟拉下来，距离近到所有人都不知道该不该继续演。

镜头：顾沉舟僵住。沈知意站在旁边，看见这场戏终于疯到自己之外。

沈知意：你们顾家平时吃饭都这么演吗？

动作：[女主演员]这句说得太像观众吐槽，现场有人真的笑了。

顾沉舟：温雅，你够了！

动作：[男主演员]推开温雅时手收得太明显，像怕售后投诉。

温雅：我够了？顾沉舟，你当着我的面吻她，现在还说我够了？

内心：她终于把体面话撕开，说出自己被当空气的羞辱。

温雅：你可以拿她刺激我，但你不能拿她赢我。

动作：[女二演员]这句演得过甜，像在和前任吵架又怕镜头不好看。

沈知意：那你冲他去，别冲我。我今天只是路过你们顾家的精神病院。

老赵：这句不是本里的，但先留。土得很准。

温雅：如果不是你，今晚站在这里的人就是我！

沈知意：那你站啊。我又没跟你抢。

镜头：顾沉舟想重新控场，但他的脸已经写着“这个场谁也控不住”。

顾沉舟：都给我闭嘴。

动作：[男主演员]喊得像夜场控麦，威严没有，音量有了。

### 幕 05｜顾家选女主人

镜头：晚宴厅入口终于有人被吵声引出来。顾家长辈站在光里，像一座旧规矩成了精。

顾家长辈：今晚谁能留下，谁就是顾家的女主人。

动作：[助理演员]说这句时太顺，顺到不像豪门长辈，像物业宣布车位归属。

沈知意：你们顾家现在连人都要这么选？

温雅：为什么不能选？我陪了他三年，凭什么不能选？

动作：[女二演员]“三年”两个字说得太用力，像真的替自己算过一笔账。

顾家长辈：顾家的体面不能没人撑，谁能撑住，谁就留下。

内心：长辈不认为这是侮辱，而认为这是给位置。

动作：[助理演员]演到“体面”时看了一眼自己的袖口，像突然想起这件西装是租的。

顾沉舟：够了，这不是你们能决定的事。

沈知意：你现在拦他们，是怕他们羞辱我，还是怕他们把你的心思说太明白？

动作：[女主演员]这句打得很准，但情绪又有点太爽，像提前进入大结局。

老赵：收一点，第一集还没完，不能现在就赢光。

温雅：那你说啊，顾沉舟，你到底要谁留下？

镜头：人群外传来急促脚步声。

动作：周特助冲进来，像比顾沉舟本人更急。

### 幕 06｜周特助下跪

动作：周特助突然冲上来，眼圈发红。

动作：[助理演员]跑得太真实，差点撞到灯架，摄影师下意识伸手护机器。

周特助：顾总！别闹了！老爷子撑着一口气，就是想见少夫人回家！

动作：[助理演员]喊“顾总”时破音了，破得像真的被拖欠工资三个月。

沈知意：你叫谁少夫人？

周特助：少夫人，顾家不能没有您！

动作：原剧本写“扑通一声跪下”。[助理演员]跪到一半停住，膝盖在空中犹豫。

老赵：他在算地板硬不硬。

动作：[助理演员]最后选择扶着桌角半跪，表情比真跪还悲壮。

周特助：这三年顾总没睡过一个安稳觉，老爷也没真正高兴过一天！

动作：[助理演员]越说越像在讲自己的三年，眼圈真的红了。

沈知意：所以呢？顾家睡不好，也要我负责？

镜头：全场没人敢接话。顾沉舟僵住，温雅脸色越来越白。

内心：沈知意看着半跪在地上的周特助，像终于看懂顾家这套秩序有多荒唐。

周特助：您不回来，顾家真的就散了。

动作：[助理演员]这句说得太诚恳，诚恳到现场所有人都开始尴尬。

### 幕 07｜只有您和温小姐的嘴

周特助：少夫人，顾总这三年碰过的女人，只有您和温小姐的嘴！

镜头：全场彻底安静，不是震惊，是那种“这也能说”的空白。

动作：[助理演员]说完自己也愣住，像终于意识到这句话不能在正常人类社会播出。

顾沉舟：周特助。

动作：[男主演员]这声叫得很轻，像客人点了不该点的酒。

温雅：周特助，你疯了吗？！

内心：温雅被公开放在和沈知意同一张荒唐表格上。

沈知意：你们顾家，是真没一个正常人了。

动作：[女主演员]这句说得非常像互联网热评，老赵在监视器后点了点头。

周特助：少夫人，您要是不回去，顾家今晚就真的散了！

动作：[助理演员]把“散了”说成了“我也散了”，情绪完全跑到自己身上。

周特助：温小姐，您不是顾家的人。可少夫人，是老爷亲口认下的。

动作：[女二演员]听完这句，表情先空白，再发狠，像直播间突然断网前最后一帧。

动作：[女主演员]说到“当众下跪”时看了半跪的周特助一眼，差点笑场，又硬生生压回去。

动作：[男主演员]“一切都是你的”说得像在推销酒水套餐，诚恳得离谱。

动作：现场安静下来。这一次不是尴尬，是这场土味豪门戏突然撞到了一点真的东西。

### 幕 09｜真够下贱的

顾沉舟：那你说。只要你开口，我今天就把整个顾家掀了，也把你请回去。

动作：[男主演员]“请”字说得太重，像终于意识到前面一直不是请，是拖。

## 后续实现建议

### 第一阶段：先让默认开拍有演员戏剧感

改动范围：

- `src/lib/gameTypes.ts`：新增 `PerformanceProfile` / `ActorPerformanceCue`。
- `src/lib/gameData.ts`：给每个演员补充表演触发规则，或从现有 `lossDirection`、`uncontrolledPoint` 派生。
- `src/lib/prompts.ts`：要求 draft 每幕显式使用演员表演层。
- `src/lib/mockAI.ts`：mockDraft 也要根据 casting 插入演员偏移行。

目标：不使用工具时，玩家也能明显感到“这帮神人真的在演，而且演歪了”。

### 第二阶段：让工具改变演员状态

改动范围：

- `InterventionResponse.actorStateDelta` 扩展为更明确的压力、偏移、配合度变化。
- 每次工具干预后，将某个演员的后续偏移变强或变弱。
- 后续幕修订时必须读取这些状态，而不是只读剧情事实。

目标：玩家不是在改剧本，而是在管理一组正在把剧本演歪的人。

### 第三阶段：让片场状态反过来压迫演员

改动范围：

- 预算低：演员更计较动作、加班、盒饭。
- 爆相高：演员更容易抢戏。
- 体面低：演员更容易放飞，豪门感变土味。
- 可控低：更多即兴台词、抢拍、笑场、走位错误。

目标：片场状态不是数字装饰，而是决定下一幕事故密度。
