# AI 工具使用过程说明

本作品在开发和运行中使用 AI 作为“片场引擎”，但固定剧本骨架、角色关系、核心幕结构由工程文件预设，AI 不负责任意改写主线剧情。

## 游戏内 AI 使用

1. 入组说服阶段  
   AI 根据玩家选择的素人演员、分配角色和说服关键词，生成演员表面回应与隐藏入组心态。

2. 开拍前 9 幕生成  
   AI 只生成演员现场反应，不修改剧本台词。后端会过滤 AI 返回结果，只接受带演员反应标记的 beat。

3. 导演工具干预  
   玩家使用喊卡、改词、加鸡腿、导演示范时，AI 会读取当前句、演员心态、片场状态和事实账本，把玩家操作续写成片场发生的现场状况。

4. 样片结算  
   AI 汇总本局选角、说服词、工具干预、事故链和最终数值，生成样片标题、风味标签、名场面描述和结算评语。

## 开发中 AI 辅助

开发过程中使用 AI 辅助完成：

- 玩法机制梳理；
- prompt 设计与调试；
- demo 演示脚本整理；
- 前端交互与样式问题排查；
- Vercel 日志调试与问题定位。

工程中的 API key、部署 token 和本地环境变量不随提交包提供。
