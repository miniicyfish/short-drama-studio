## 工程运行方式

本工程是 Next.js 项目。若需要本地运行：

```bash
npm install
cp .env.example .env.local
npm run dev
```

然后打开：

```txt
http://localhost:3000
```

`.env.local` 需要自行填入 OpenAI-compatible API 配置：

```env
AI_API_KEY=your_api_key_here
AI_BASE_URL=https://your-openai-compatible-endpoint/v1
AI_MODEL=your_model_name
```

## 核心玩法说明

玩家扮演短剧片场导演，在短剧爆火前十年，用一群素人演员拍出第一集样片。游戏会先根据选角和说服词生成演员“入组心态”，再在开拍前生成 9 幕拍摄稿。

AI 主要用于：

- 生成演员入组心态；
- 根据真实演员和角色关系生成 9 幕片场反应；
- 根据玩家使用的导演工具改写当前片场事实；
- 在结尾回收本局演员、工具和片场事故，生成样片结算。

导演工具包括：喊卡、改词、加鸡腿、导演示范。工具不是简单数值按钮，而是一次带语境的片场行为，会影响当前句、当前幕剩余内容和后续事实账本。
